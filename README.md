# customer-trends-data-analysis-SQL-Python-PowerBI

🛍️ Customer Shopping Behavior Analysis
📌 Project Overview

This project analyzes customer shopping behavior using real-world transactional data.
The goal is to understand how customers shop, what they prefer, and how businesses can improve sales and customer experience.

🎯 Business Problem

A retail company wants to:

Understand customer purchasing patterns
Identify factors affecting buying decisions
Improve customer engagement and loyalty

📊 Dataset Details
📦 Total Records: 3,900
📑 Columns: 18
👥 Includes:
Customer demographics (Age, Gender, Location)
Purchase details (Product, Category, Amount)
Behavior data (Discounts, Reviews, Frequency)
⚠️ Missing Values: Only in Review Rating column

🛠️ Tools & Technologies Used
🐍 Python (Pandas, NumPy)
🗄️ SQL (PostgreSQL)
📊 Power BI (Dashboard)
📁 Jupyter Notebook
🔍 Project Workflow

1. Data Preparation (Python)
Loaded dataset using pandas
Cleaned missing values
Created new features like:
Age Groups
Purchase Frequency

2. Data Analysis (SQL)

Performed business analysis such as:

Revenue by gender
Top-rated products
Customer segmentation (New, Returning, Loyal)
Impact of discounts and shipping

3. Data Visualization (Power BI)

Created an interactive dashboard showing:

Sales by category
Revenue by age group
Subscription insights
Customer distribution
